Why is fragment needed in android application? How can a fragment be added to an activity?

Why Fragments are Needed:

Fragments are a way of modularizing your UI. They allow you to break up your activity's layout into reusable components. This is especially useful for creating flexible UIs that can adapt to different screen sizes.

Adding a Fragment to an Activity:

You can add a fragment to an activity in two ways:

  • Statically: You can add a fragment to an activity's layout file using the <fragment> tag.
  • Dynamically: You can add a fragment to an activity at runtime using the FragmentManager.

What is menu? Develop an android application to create a popup menu having menu item "Edit", "Copy", "Paste", "Redo" and "Undo.".

What is a Menu:

A menu is a UI component that displays a list of options to the user. Android provides several types of menus, including options menus, context menus, and popup menus.

Popup Menu Example:

1. Create a menu resource file (e.g., res/menu/popup_menu.xml):


<?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item
        android:id="@+id/menu_edit"
        android:title="Edit" />
    <item
        android:id="@+id/menu_copy"
        android:title="Copy" />
    <item
        android:id="@+id/menu_paste"
        android:title="Paste" />
    <item
        android:id="@+id/menu_redo"
        android:title="Redo" />
    <item
        android:id="@+id/menu_undo"
        android:title="Undo" />
</menu>

2. In your activity, create and show the popup menu:


Button button = findViewById(R.id.show_popup_menu_button);
button.setOnClickListener(v -> {
    PopupMenu popup = new PopupMenu(this, v);
    popup.getMenuInflater().inflate(R.menu.popup_menu, popup.getMenu());
    popup.setOnMenuItemClickListener(item -> {
        // Handle menu item clicks
        return true;
    });
    popup.show();
});

What are the uses of fragments? How fragments are differing from activity?

Uses of Fragments:

  • Creating flexible and reusable UI components.
  • Building multi-pane layouts for tablets.
  • Managing the UI of a single screen in a more modular way.

Difference between Activity and Fragment:

An activity is a single, focused thing that a user can do. A fragment is a portion of a user interface in an activity. An activity can have multiple fragments, but a fragment must be hosted by an activity.
