What do you mean by storyboard? Write a swift program to find the largest and smallest number among three numbers using a ternary operator.

Storyboard:

A storyboard is a visual representation of the user interface of an iOS application, showing screens of content and the connections between those screens. It allows you to design and layout your UI without writing any code.

Swift Program for Largest and Smallest:


let a = 10
let b = 20
let c = 5

let largest = a > b ? (a > c ? a : c) : (b > c ? b : c)
let smallest = a < b ? (a < c ? a : c) : (b < c ? b : c)

print("Largest: \(largest)") // Output: Largest: 20
print("Smallest: \(smallest)") // Output: Smallest: 5

What do you mean by View hierarchy in iOS programming? Write a swift function to calculate the factorial of any positive integer.

View Hierarchy:

The view hierarchy is a tree-like structure of views that make up the user interface of an iOS application. At the root of the hierarchy is the window object, and each view in the hierarchy has a single superview and zero or more subviews.

Swift Function for Factorial:


func factorial(of n: Int) -> Int {
    return n == 0 ? 1 : n * factorial(of: n - 1)
}

print(factorial(of: 5)) // Output: 120
