How is List View differ from Recycler View? Explain with an example.

ListView vs. RecyclerView:

The main difference between ListView and RecyclerView is that RecyclerView is more efficient and flexible. RecyclerView recycles views that are no longer visible to the user, which improves performance and reduces memory consumption. ListView, on the other hand, creates a new view for each item in the list, which can be inefficient for long lists.

ArrayAdapter vs. CursorAdapter:

An ArrayAdapter is a simple adapter that can be used to display a list of items from an array or a list. A CursorAdapter is used to display data from a database query.
